# 常见简化假设库

Stage 1（设定书）和 Stage 2（假设台账）的弹药库。

**用法**：建模时对着这份清单过一遍，把你**实际用到的**假设找出来，连同它的**成立判据**一起抄进假设台账。判据一栏写的是**不等式**——这是刻意的：假设台账里写"通常成立"等于没写，写 $r \gg L$ 才能代入数字验证。

**注意**：这份清单不全，也不可能全。它的作用是**提醒你去找那些你没意识到自己在用的假设**——最危险的假设永远是那些你觉得"这还用说吗"的。

---

## 一、几何简化

| 假设 | 成立判据 | 失效场景 |
|---|---|---|
| **点偶极子近似**（磁体/电偶极子当点源） | $r \gg L_{source}$（观察距离 ≫ 源尺寸）。工程上 $r > 3L$ 时误差 <10%，$r > 5L$ 时 <3% | 磁体贴近导体壁、两磁体近距离相互作用。**磁阻尼题的重灾区**：管壁离磁体只有几 mm，而磁体长 10 mm——$r\gg L$ 根本不成立 |
| **无限长**（长杆/长管/长螺线管） | $L \gg R$，且关注点距端点 $\gg R$。螺线管中心场误差 $\sim (R/L)^2$ | 短螺线管、边缘效应主导时 |
| **薄壁/薄膜** | $t \ll R$（厚度 ≪ 曲率半径）；且对涡流问题还要 $t \ll \delta$（趋肤深度） | 厚壁管（涡流题！）、厚膜 |
| **轴对称** | 几何、初条件、驱动全都轴对称 | 有偏心、有倾斜、有横向扰动时。**注意：轴对称的系统可能自发失稳到非轴对称**（如射流破碎） |
| **一维/二维退化** | 忽略的那个方向上梯度 $\ll$ 保留方向 | 端部效应、边界层 |
| **刚体** | 弹性变形 $\delta \ll L$，即 $F/(EA) \ll 1$；且驱动频率 $\ll$ 最低弹性本征频率 | 碰撞（接触时间由弹性决定！）、高频驱动 |
| **质点** | 转动惯量效应可忽略：不涉及转动，或 $I\omega^2 \ll mv^2$ | 滚动、进动、任何有力矩的问题 |

## 二、材料与本构

| 假设 | 成立判据 | 失效场景 |
|---|---|---|
| **均匀磁化**（$\boldsymbol M$ 处处相同） | 硬磁材料（NdFeB、铁氧体）在**外场 $\ll H_c$**（矫顽力）时成立，是很好的近似。软磁材料（铁、坡莫合金）**完全不成立**——它的 $M$ 由外场决定 | 软磁材料；硬磁体处在接近 $H_c$ 的反向场中；温度接近居里点 |
| **线性磁化** $B = \mu H$ | 未饱和：$H \ll H_{sat}$。铁在 $B \gtrsim 1.5\,\mathrm{T}$ 饱和 | 铁磁材料大场；有磁滞时（磁滞本身是非线性+历史依赖，$\mu$ 都没定义） |
| **忽略退磁场** | 只在细长样品沿长轴磁化时可以（退磁因子 $N \to 0$）。球体 $N = 1/3$，薄片垂直磁化 $N \to 1$——**这时退磁场和外场同量级，绝不能忽略** | 短粗磁体、薄片垂直磁化 |
| **不可压缩流体** | $Ma = v/c_s \ll 1$（工程上 $Ma < 0.3$，密度变化 <5%）。液体几乎总成立 | 高速气流、声学（声波的本质就是可压缩性！） |
| **牛顿流体** ($\tau = \eta\dot\gamma$) | 水、空气、甘油、多数纯液体成立 | 淀粉水（剪切增稠）、番茄酱（剪切变稀）、聚合物溶液、血液、沙子。**IYPT 特别爱出非牛顿题**——看到"cornstarch""oobleck"立刻警觉 |
| **各向同性** | 非晶、多晶无织构 | 单晶、纤维材料、层压板、木头 |
| **线弹性（胡克定律）** | 应变 $\epsilon \ll$ 屈服应变（金属 $\sim10^{-3}$，橡胶可到 $\sim 1$） | 大变形、塑性、橡胶（超弹性） |
| **常物性**（$\rho, \eta, \kappa, c_p$ 与 T 无关） | 温差小：$\Delta T \cdot \frac{1}{X}\frac{dX}{dT} \ll 1$。**水的黏度每升温 20°C 减半**——热学题里这个假设很容易破 | 大温差；黏度对温度极敏感的液体（糖浆、油） |
| **理想气体** | 远离临界点，$p \ll p_c$，$T \gg T_c$ | 近临界、高压、水蒸气近饱和 |

## 三、场与电磁

| 假设 | 成立判据 | 失效场景 |
|---|---|---|
| **准静态电磁**（忽略推迟、忽略位移电流） | 系统尺寸 $L \ll c\tau$（$\tau$ = 特征时间）；等价于 $L \ll \lambda$ | 高频、天线、辐射问题 |
| **均匀外场** | 场源尺寸 $\gg$ 样品尺寸，或样品处在亥姆霍兹线圈中心区 | 靠近磁体、靠近电极边缘 |
| **趋肤深度可忽略**（电流均匀分布） | $t \ll \delta = \sqrt{2\rho/(\mu\omega)}$。铜在 50 Hz 时 $\delta \approx 9\,\mathrm{mm}$，1 kHz 时 $\approx 2\,\mathrm{mm}$ | 高频、厚导体。**涡流题必查** |
| **忽略自感**（涡流回路的 $L\,dI/dt$） | 磁 Reynolds 数 $Re_m = \mu\sigma v L \ll 1$；等价于磁扩散时间 $\tau_m = \mu\sigma L^2 \ll$ 运动时间 | 高电导率（铜！）、大尺寸、快运动 |
| **忽略磁滞** | 硬磁体作为场源时可以；软磁作为磁路时**不可以** | 变压器铁芯、软磁材料循环磁化 |
| **导体表面 $E_\parallel = 0$** | 静电平衡，或良导体 | 有电流的电阻材料内部 |
| **忽略涡流反作用场**（一阶近似） | $Re_m \ll 1$，感应电流产生的场 ≪ 原场 | 强磁阻尼、液态金属 |

## 四、动力学与热力学

| 假设 | 成立判据 | 失效场景 |
|---|---|---|
| **准静态 / 稳态** | $\tau_{relax} \ll \tau_{drive}$——**必须把两个时间尺度都估出来比一比**，不许口头声称 | 快速驱动、共振附近 |
| **小角近似** $\sin\theta\approx\theta$ | 误差：$10°$→0.5%，$30°$→4.5%，$60°$→**17%**，$90°$→**57%**。看你能容忍多少 | 大摆角。**注意：小角近似 + 扫描到 90° 的图 = 同时犯 P7 和 P3** |
| **线性阻尼** $F = -bv$ | $Re \ll 1$（Stokes 区） | $Re \gtrsim 1$，这时阻力 $\propto v^2$ |
| **纯滚动**（无滑动） | 需要的摩擦力 $\le \mu_s N$——**要真的验证这个不等式**，不能假定 | 打滑（急加速、光滑面、大倾角） |
| **无摩擦** | $\mu N \ll$ 主项——给数字 | 几乎所有真实系统里都不成立，慎用 |
| **弹性碰撞** | 恢复系数 $e \approx 1$；实测钢球-钢板 $e\approx0.9$，橡胶 $\approx0.8$，黏土 $\approx0$ | 大多数碰撞 |
| **瞬时碰撞** | 接触时间 $\tau_c \ll$ 运动时间尺度 | 软材料碰撞、需要研究接触过程本身时（Hertz 接触） |
| **绝热** | $\tau_{thermal} = L^2/\alpha \gg \tau_{process}$。**声波是绝热的**（快） | 慢过程 |
| **等温** | $\tau_{thermal} \ll \tau_{process}$。毛细上升、缓慢压缩是等温的 | 快过程。**绝热和等温搞反，声速差 $\sqrt{\gamma}\approx1.18$ 倍** |
| **忽略热辐射** | $\sigma_{SB}T^4 \ll$ 其他传热项。室温下辐射常常和自然对流同量级——**不能想当然忽略** | 高温（$\propto T^4$，升温很快就主导）、真空中 |

## 五、环境

| 假设 | 成立判据 | 失效场景 |
|---|---|---|
| **忽略空气阻力** | $\frac12\rho_{air}C_d A v^2 \ll$ 主项——给数字，并做**扫描端点检查** | 高速、大面积、轻物体（羽毛、气球、纸片） |
| **忽略空气浮力** | $\rho_{air}/\rho_{object} \ll$ 你要的精度。空气 $1.2\,\mathrm{kg/m^3}$ vs 水 $1000$ → $10^{-3}$，通常可忽略；但对**气球、泡沫**完全不可忽略 | 低密度物体、高精度测量 |
| **重力均匀** | 尺度 $\ll R_{Earth}$ | 几乎总成立 |
| **忽略地球自转（科氏力）** | $2\omega v \ll a$。$\omega_{Earth}=7.3\times10^{-5}\,\mathrm{s^{-1}}$，通常可忽略；但**傅科摆、长时间尺度的题不行** | 傅科摆、大气/海洋、长时程 |
| **温度恒定** | 过程中的自加热 $\Delta T$ 小到不改变物性——**耗散功率大时要查**（涡流会让管子发热、黏性会让流体升温） | 大功率耗散、长时间运行 |
| **忽略蒸发** | 蒸发时间 $\gg$ 实验时间。**小液滴的实验里几乎总是不成立**（毫米液滴在室温下几分钟就明显变小） | 液滴题、开放液面、长时间 |
| **忽略振动/气流干扰** | 实验条件——这是 **Setting**（设定），不是 Assumption。写进设定书 | — |

## 六、几个"看起来无害、实际致命"的假设

单独拎出来，因为它们最常被默默使用而不自知：

1. **"稳态已经建立"** —— 涡流、温度场、流场的建立都需要时间。你的系统跑了足够久吗？（P9）
2. **"一维处理就够了"** —— 边界层、端部效应、边缘场往往就是全部的物理。
3. **"参数在扫描范围内不变"** —— $C_d(Re)$ 随 $Re$ 变（阻力危机时骤降一半！）；$\eta(T)$ 随温度变；$\mu(H)$ 随场变。**把变量当常数是隐式假设**。
4. **"可以线性叠加"** —— 只有线性系统才行。有摩擦、有湍流、有磁滞、有接触，就不能叠加。
5. **"实验/文献里那个数适用于我的情况"** —— 文献的 $C_d$ 是在什么 $Re$ 下测的？文献的表面张力是在什么温度、什么纯度下测的？（P15）
6. **"这个效应是二阶小量，可以忽略"** —— 除非你算了。二阶小量在共振附近、在长时间积累后（久期项）、在主项恰好抵消时，都会变成主项。

## 分级怎么打

| 分级 | 判据 |
|---|---|
| `SAFE` | 即使这条假设完全不成立，主结论（幂律指数、单调性、量级）**不变**。或者判据不等式满足得非常宽松（差 2 个数量级以上） |
| `LOAD-BEARING` | 主结论**依赖**这条假设。它成立（判据满足），但如果不成立，结论会变。**必须在正文里说清"如果它不成立会怎样"** |
| `RISKY` | 判据**只是勉强满足**（同量级、差不到一个数量级），或者你**无法确定**它是否成立。**必须生成一个数值验证任务交给 Skill 2** |

**一条经验**：如果你在写判据不等式的时候，发现自己得把参数代进去才知道成不成立——那它至少是 `LOAD-BEARING`。如果代进去发现两边只差 3 倍——那是 `RISKY`。

`RISKY` 不是坏事，`RISKY` 但没标出来才是。标出来的 RISKY 假设 + 一个验证它的数值实验 = **满分答案**；藏起来的 RISKY 假设 = 被 Opponent 一击致命。
