# 建模操作手册

这份不是概念科普，是**操作步骤**。四个工具：量纲分析、量级估计、机制预算、极限检验。每个都给"怎么做"和"做到什么程度算做完"。

---

## 一、量纲分析（Buckingham Π）

**为什么先做它**：在写下任何微分方程之前，量纲分析就能告诉你"答案必然长什么样"。它便宜（十分钟）、几乎不会错（只靠量纲）、而且**能在建模前就抓出错误**——如果你的详细模型给出的标度律和量纲分析矛盾，一定是模型错了。

IYPT 的题目八成是"研究 Y 对 X 的依赖"。**Π 定理直接告诉你这个依赖的骨架长什么样。** 不做它就是把分数扔掉。

### 步骤

1. **列出所有相关物理量**（含待求量），写出各自的量纲。这一步是唯一需要物理直觉的地方，也是唯一会出错的地方——**漏掉一个量，结论就废了**。列完之后回头问："我漏了什么？粘度？表面张力？重力？系统的某个尺寸？"

2. **数：** $n$ = 物理量个数，$k$ = 独立的基本量纲个数（通常是 M, L, T，加上电磁题的 I 或 热学题的 Θ）。

3. **Π 定理**：可以构造 $n - k$ 个独立的无量纲组 $\Pi_1, \dots, \Pi_{n-k}$。

4. **构造它们**：选 $k$ 个"重复量"（要能张成所有量纲，且不能自己组成无量纲组），把剩下每个量分别和重复量组合成无量纲组。

5. **结论**：$\Pi_1 = f(\Pi_2, \dots, \Pi_{n-k})$。

### 关键的三个技巧

**技巧 1：只剩一个 Π 时，你就白拿了一个完整的定律。**
如果 $n - k = 1$，那么 $\Pi_1 = \text{const}$——待求量的形式被**完全确定**，只差一个数字系数。

> 例：单摆周期。物理量：$T, l, g, m$（$n=4$），量纲 M, L, T（$k=3$）。$n-k=1$。唯一的无量纲组是 $T\sqrt{g/l}$。所以 $T = C\sqrt{l/g}$，**且 $T$ 与质量 $m$ 无关**——这个非平凡结论完全由量纲得出，一行方程都没写。（$C = 2\pi$ 需要真解方程。）

**技巧 2：$n-k=2$ 时，你得到一个"数据坍缩"。**
$\Pi_1 = f(\Pi_2)$：意味着如果你把实验数据画成 $\Pi_1$ vs $\Pi_2$，**所有不同参数的曲线会坍缩到同一条线上**。这是 IYPT 报告里杀伤力最大的一张图——它证明你抓住了正确的无量纲组。**务必在 `figures[]` 里要求 Skill 2 画这张坍缩图。**

**技巧 3：幂律指数是零自由参数的预测。**
如果量纲分析给出 $v_t \propto B^{-2}$，那么"指数是 $-2$"是一个**不含任何可调参数**的预测。实验/仿真测出的斜率是 $-2.0$ 还是 $-1.4$，直接判模型生死。这比"曲线拟合得很好"有力得多（见错误模式 P12）。

### 做到什么程度算做完

- [ ] 列出了所有物理量，且**明确写出"我认为不相关的量及其理由"**（这是漏项的唯一防线）
- [ ] 给出了全部 $n-k$ 个 Π 组，并**给它们物理解释**（这个组是 Re？是 Bond 数？是某两个时间尺度的比？）
- [ ] 写出了预期的标度律 / 坍缩形式
- [ ] 如果无量纲组是已知的（Re, We, Bo, Fr, Pe, Ma, Ra…），**认出来并说出它的物理意义**

### 常见的无量纲组速查

| 组 | 定义 | 物理意义 | 什么时候你该想到它 |
|---|---|---|---|
| Reynolds $Re$ | $\rho v L/\eta$ | 惯性 / 黏性 | 任何有流体和运动的题 |
| Weber $We$ | $\rho v^2 L/\gamma$ | 惯性 / 表面张力 | 液滴、射流、飞溅 |
| Bond $Bo$ | $\rho g L^2/\gamma$ | 重力 / 表面张力 | 液滴形状、毛细、浮起 |
| Capillary $Ca$ | $\eta v/\gamma$ | 黏性 / 表面张力 | 涂布、拉丝、移动接触线 |
| Froude $Fr$ | $v/\sqrt{gL}$ | 惯性 / 重力 | 波、船、跳跃、水跃 |
| Mach $Ma$ | $v/c_s$ | 可压缩性 | $Ma>0.3$ 时不可压假设失效 |
| Péclet $Pe$ | $vL/D$ | 对流 / 扩散 | 传热传质 |
| Prandtl $Pr$ | $\nu/\alpha$ | 动量扩散 / 热扩散 | 自然对流 |
| Rayleigh $Ra$ | $g\beta\Delta T L^3/(\nu\alpha)$ | 浮力驱动对流强度 | 热对流失稳（$Ra_c\approx1708$） |
| 磁 Reynolds $Re_m$ | $\mu\sigma v L$ | 磁对流 / 磁扩散 | 涡流、磁阻尼 |
| Strouhal $St$ | $fL/v$ | 振荡 / 对流 | 涡街脱落（$St\approx0.2$） |
| Deborah $De$ | $\tau_{relax}/\tau_{obs}$ | 弹性 / 黏性 | 非牛顿流体、"这东西算固体还是液体" |

## 二、量级估计（Fermi）

**规则：任何"忽略"都必须以一个数字结束。** 口头忽略是审稿人的 P2，也是模型和实验对不上时最常见的元凶。

### 步骤

1. 写出被略去项的**表达式**（不是"空气阻力"，是 $\frac12\rho_{air}C_d A v^2$）；
2. 写出主项的表达式；
3. **代入设定书里的具体数值**（这是为什么 Stage 1 必须先把参数定死——没有数值就做不了量级估计）；
4. 算出**比值**，给出数字；
5. 判定：$\lesssim 10^{-2}$ → negligible；$10^{-2}$ 到 $10^{-1}$ → correction（进 Model-1）；$\gtrsim 10^{-1}$ → dominant（必须进 Model-0）。

### 然后做扫描端点检查（这一步最常被跳过）

比值不是一个数，**是参数的函数**。把它写成函数，代入扫描范围的**两个端点**：

> 基准值下 $F_{air}/F_{mag} = 1.5\times10^{-4}$ ✓ 可忽略
> 但 $F_{air}/F_{mag} \propto v^2/B^2$，而扫描 $B$ 从 0.5 T 降到 0.05 T 时 $v_t$ 上升 100 倍
> ⇒ 在 $B = 0.05\,\mathrm{T}$ 端，比值 $= 1.5\times10^{-4} \times 100^2 / 0.01^{-1}\dots$ 算一下 → **$O(1)$，不可忽略！**

**扫描范围的一端可忽略，不代表另一端可忽略。** 这是审稿模式 P3，也是"曲线的右半段整段是错的"的根源。

## 三、机制预算（Mechanism Budget）

把所有候选物理机制排成一张表，逐个量级估计，明确它的去向。**这张表是分析的骨架**，Model-0 该留什么、Model-1 该加什么，全由它决定。

### 候选机制清单（挨个问"这题里有没有它"）

| 类别 | 机制 | 典型量级 |
|---|---|---|
| 力学 | 重力 | $mg$ |
| | 惯性 | $ma$, $\rho v^2 L^2$ |
| | 弹性 | $EL\delta$, $k x$ |
| | 摩擦（干） | $\mu N$ |
| 流体 | 黏性 | $\eta v L$ |
| | 压差 | $\Delta p\, L^2$ |
| | 表面张力 | $\gamma L$ |
| | 浮力 | $\rho g V$ |
| 电磁 | 静电 | $q E$, $\epsilon_0 E^2 L^2$ |
| | 磁（力） | $BIL$, $\nabla(\boldsymbol m\cdot\boldsymbol B)$ |
| | 涡流耗散 | $\sigma \dot B^2 L^5$ |
| 热 | 传导 | $\kappa \Delta T L$ |
| | 对流 | $h\Delta T L^2$ |
| | 辐射 | $\sigma_{SB} T^4 L^2$ |
| | 相变潜热 | $\rho L_v V$ |

### 表格格式（直接抄进 `01-analysis.md`）

| 机制 | 表达式 | 代入数值 | 量级 (N 或 J) | 与主项之比 | 去向 |
|---|---|---|---|---|---|
| 重力 | $mg$ | $0.02\times9.8$ | $2.0\times10^{-1}$ | 1（主项） | **Model-0** |
| 磁阻尼 | $\sigma \dots$ | … | $2.0\times10^{-1}$ | 1 | **Model-0** |
| 空气阻力 | $\frac12\rho C_d A v^2$ | … | $3\times10^{-5}$ | $1.5\times10^{-4}$ | 忽略（但见扫描端点检查） |
| 管壁摩擦 | $\mu N$ | 无接触 | 0 | 0 | 忽略（几何上不接触） |

**硬规则**：
- "去向"列里每一个"忽略"，**都必须在"与主项之比"列里有数字**；
- 每个被忽略的项都要做扫描端点检查；
- 如果某个机制你**没考虑**，也要列出来并说"为什么它不在这题里"——空着不写就是漏项。

## 四、极限检验（Sanity Battery）

模型推完之后、交给审稿人之前，**自己先跑这六项**。每一项都便宜，每一项都真的能抓到错。

### 1. 量纲齐次性
逐式检查。**每一个**编号公式。包括中间步骤引入的常数。

### 2. 极限行为
把参数推到极端，看结果是否退化成**你已知的东西**：
- 阻尼 → 0：回到无阻尼简谐/自由落体了吗？
- 质量 → 0 或 → ∞：行为合理吗？
- 尺寸 → 0：回到点粒子结果了吗？
- $t \to \infty$：耗散系统必须停下/达到稳态。
- 某个耦合 → 0：两个子系统必须解耦。

**这一项抓错误的效率极高**——因为极限情形通常有已知答案，对不上就是错了。

### 3. 对称性与守恒
- 系统有的对称性，解也该有（除非是自发对称破缺——那要明说）；
- 能量：无耗散时守恒；有耗散时**单调减**（$dE/dt \le 0$，符号！）；
- 动量/角动量：有无外力矩？

### 4. 数值可信度
把设定书的数值代进最终公式，**算出一个具体的数**，问：
- 这个速度/温度/频率**物理上可能吗**？（不能超音速、不能超过环境温度上限、不能是负的绝对温度）
- 和日常经验对得上吗？（磁铁在铜管里下落，终速应该是"肉眼可见地慢"，$O(1)\,\mathrm{cm/s}$ 量级——如果你算出 5 m/s，模型错了）

**这条常常在五分钟内就把一个错误模型枪毙掉。**

### 5. 已知特例
你的模型在某个特殊参数下，应该退化成教科书上的经典结果吗？如果是，检查它真的退化了。

### 6. 反向常识检查
把结论念给一个懂物理的人听："管壁越厚，磁铁掉得越快"——**等一下，这听着就不对**。感觉不对的时候，通常是真的不对。

## 五、什么叫"充分的分析"

不是"写得长"。是这五件事都做到了：

1. **每一条简化都被显式记账**，且带**不等式形式**的成立判据（不是"通常成立"）；
2. **每一个忽略都有数字**，且做过扫描端点检查；
3. **有一个不含自由参数的预测**（幂律指数、无量纲组的临界值、坍缩曲线）——这是模型可被证伪的地方，也是分数所在；
4. **模型分层**：Model-0 能手算、能一眼看懂物理；Model-2 才上数值。上来就 FEM 硬解，说明你没理解这个现象；
5. **知道自己什么时候会错**：说得出"如果 XX 假设不成立，我的结论会变成 YY"。
