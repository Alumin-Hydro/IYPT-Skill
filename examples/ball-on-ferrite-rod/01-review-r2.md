## 命中的洞

### H1' · [MAJOR] r1 的 H1 只关了一半：「$1f$ 主导」无量级支撑，且 $1f{+}2f$ 让 BC 判据重新二义 — 命中 P8 / P17 / P18

- **位置**：S-7 / §4 M-1「加偏置 ⟹ 力**主导 $1f$**（$\propto 2H_bH_{ac}$）+ $2f$ 泛音 ⟹ 驱动在 $f{=}f_1$ 直接激发基频」；A-1 退化特征「峰在 $c/4L$=14 ⟹ 固定-自由，峰在 $c/2L$=28 ⟹ 自由-自由」；F-1 / `risky_assumption_checks[A-1]` / `criterion_matrix` 的 C1。

- **为什么崩**（实算 `§5`）：
  1. **「$1f$ 主导」没有量级。** 展开 $\lambda\propto(H_b+H_{ac}\cos\omega t)^2$：$1f$ 幅度 $\propto 2H_bH_{ac}$、$2f$ 幅度 $\propto H_{ac}^2/2$ ⟹ **$1f/2f = 4H_b/H_{ac}$**。「$1f$ 主导」要求 $H_b\gg H_{ac}/4$。**分析里 $H_b$、$H_{ac}$ 一个数都没有。** 而 S-1 选的是**软磁 Ni-Zn 铁氧体**（低矫顽、低剩磁——正是它做天线/变压器芯的原因），其剩磁给的 $H_b$ **多半偏小**；S-7 的替代方案「线圈叠直流分量」又不在参数表里（$I_0$ 只写了 AC 幅度 $0$–$0.5$ A、无 DC 偏置电流）。**⟹ 支撑整个「Explain / 为什么驱动在 $f_1$」的偏置强度，既没量化、在选定材料上还可疑。**
  2. **$1f{+}2f$ 让共振峰二义。** 扫驱动频率 $f$、力含 $f$（$1f$）与 $2f$ 分量，各自撞到本征频率时出峰：
     - **固定-自由**（$f_1{=}14$ kHz）：$1f$ 撞 $f_1$ ⟹ 峰在 $f{=}14$；$2f$ 撞 $f_1$ ⟹ 峰在 $f{=}7$。**峰集 $\{7,14\}$。**
     - **自由-自由**（$f_1{=}28$ kHz）：$1f$ 撞 ⟹ 峰在 $28$；$2f$ 撞 ⟹ 峰在 $14$。**峰集 $\{14,28\}$。**
     - **⟹ 两种 BC 都在 14 kHz 有峰。** 一个 14 kHz 的峰**同时**兼容固定-自由（$1f$）和自由-自由（$2f$）。干净的判别子是**第二个峰**（$7$⟹固定-自由、$28$⟹自由-自由），**不是「峰在 14 vs 28」**。而 A-1 退化特征、F-1、C1 全写的是后者——**这正是 r1 H1 那个 $\times2$ 换了个地方复活**：r1 是「峰应在 7 不在 14」，修订后变成「7 和 14 都有峰、14 二义」。
  3. **matrix 的 C1 把 `peak` 硬编码成本征频率**（`criterion_matrix.py` L64: `peak = model.get("f1", f_res)`；correct 直接取 `f_res`，WM1 直接取 `f1(free-free)`）。它**从不模拟含谐波的驱动扫频**，故对上面的二义**结构性看不见**——C1 证明的是「你公式里用了 $c/4L$ 还是 $c/2L$」，不是「你能否从实测扫频里把 $f_1$ 读对」。**F-1 才是真正会踩坑的地方，而 C1 不碰它。**

- **wrong-quantity 元探针（C1）**：C1 该界的是**棒本征频率**，代进去的是**驱动扫频的峰位**。二者在偏置修法下解耦：若偏置弱（$2f$ 相对强），固定-自由的显著峰落在 $f{=}7$ kHz ⟹ C1 判 $|7000/13975-1|=0.50>0.10$ ⟹ **把完全正确的固定-自由模型判死**；反过来自由-自由的 $2f$ 峰落在 14 kHz ⟹ C1 会**把自由-自由错判成固定-自由**。**门界错了量。**

- **影响**：**T-3（共振依赖）+ A-1（全分析最脆弱的 RISKY）的判别子被污染**，而 matrix 的 C1 给了「这条判据没问题」的假保证。Physics Fight 上一句「你偏置多强？$H_b/H_{ac}$ 是多少？若 $1f$ 不主导，你扫频时 7 kHz 也有峰，14 kHz 的峰分不清固定-自由还是自由-自由」——Reporter 答不上来。**不污染球动力学的数（$\Gamma,\bar h$ 用棒尖实测 $A$，与偏置无关）——崩的仍是「Explain / 共振」那半道题。**

- **可否修**：不重做 Stage 4。① 给 $H_b/H_{ac}$ 一个量级（或承认软铁氧体剩磁不足、必须外加 DC 偏置电流，并把它列进参数/扫描）；② A-1 退化特征与 F-1 改成「看**是否存在** 7 kHz（固定-自由）或 28 kHz（自由-自由）的峰」，或改测**棒尖振铃频率**（棒尖在共振处以 $f_1$ 振，与哪个谐波驱动无关——这才是免疫谐波的干净判别子，V-1 已有测振仪，只是判据写错了量）；③ C1 在 `criterion_matrix.py` 里**真的模拟一次含 $1f{+}2f$ 的扫频**，让它能被 7/14/28 的二义证伪。

### H2' · [MAJOR] C2-slope 容差从错误的误差传播里推出（$\bar h\propto A^2$ 的因子 2 丢了）；robustness 只扫 $e$、对 $A$ 失明 — 命中 P18 ①②④ / P17 ③

- **位置**：`criterion_matrix.py` L81-83 与 `matrix.json` 的 C2；`robustness_scan`；`min_detectable.offset_C2`。
- **为什么崩**（实算 `§3`,`§7`）：C2 容差 $=0.15$，`tolerance_source` 逐字写「独立测 $A$（干涉 $\sim10\%$）+ $e$（落球 $\sim5\%$）⟹ 预言侧不确定度 $\sim15\%$」。**这个推导错两处**：
  1. **$\bar h=(1{+}e)/(1{-}e)(A\omega)^2/4g\propto A^2$** ⟹ $\delta\bar h/\bar h = 2\,\delta A/A = 2\times10\% = \mathbf{20\%}$。源码把 $A$ 的 $10\%$ 当成线性流进 $\bar h$，**漏了 $A^2$ 的因子 2**。
  2. $e$ 的贡献：$\dfrac{2\,\delta e}{1-e^2}=\dfrac{2\times0.03}{0.64}=9.4\%$。**求积** total $=\sqrt{20^2+9.4^2}=\mathbf{22.1\%}$（线性 $29.4\%$）。**源码还把 $10\%+5\%$ 线性相加**——两处错叠一起，把 $22\%$ 说成 $15\%$。
- **三个后果**（都是 P18 存在的理由）：
  1. **$\varepsilon^*$ 虚高**：`min_detectable.offset_C2.eps_star = 0.156` 宣称「C2 能分辨 $\bar h$ 系数 $15.6\%$ 的偏差」。**但真实预言本底是 $22\%$——$15.6\%$ 的偏差埋在噪声里，分辨不出。** 这正是 P18 ②「$\varepsilon^*$ 是可汇报的物理结论」的反面：**这里它是一句被高估的假结论。**「我们能分辨 $\bar h$ 到 $15.6\%$」不是真话。
  2. **过紧的门会误杀正确模型**：实测 $\bar h$ 按诚实的 $\pm22\%$ 散布，门却卡在 $\pm15\%$ ⟹ **约三成正确测量落在门外、被判 FAIL**——下游会去「修」一段本来正确的物理。更具体：分析自己（A-3 / r1 的 H5）承认 $\tau_c/T\sim0.16$–$0.20$ 的有限接触时间会给有效 $e$ / $\bar h$ 斜率一个 **Model-1 修正**（量级未算，但плаusibly $\sim20\%$）——若 Skill 2 照 §5 建 Model-2，它对 Model-0 的偏离就可能 $>15\%$，**被 C2 当错模型毙掉**。
  3. **`robustness_scan` 对主导不确定度失明**：它**只扫 $e$**（L165 `for de in [...]`），报 `delta_max=0.07375`、`margin=1.475`——**从不扰动 $A$**。而 $A$ 的 $10\%$（$\to\bar h$ 的 $20\%$）才是压垮 $15\%$ 门的那一项。**「不误杀」这一列的 PASS，是在一条它根本没测的自由度上宣布的。** 顺带：$1.475\times$ 的裕度本身就薄（electrical-damping 死在 $1.7\times$）。

- **wrong-quantity 元探针（C2）**：C2 界的是**绝对 $\bar h$**（正确——这是唯一能抓「形状全对、系数偏了」的量，WM5 靠它）。问题不在界错量，在**容差的来源算错了**、且 robustness 扫错了自由度。

- **影响**：C2 是 Skill 2 验收 $\bar h$ 的**门阈值**（P17 ③：门阈值错 ⟹ MAJOR）。它同时让 `matrix` 报出一个虚高的分辨率、并埋下对 Model-1 修正的伪失败。
- **★ 机械门为什么没抓到**：`check_analysis.py` 有 `CRIT-TOLERANCE-UNSOURCED`（查「容差有没有来源字符串」）和 `CRIT-NO-ROBUSTNESS`/`CRIT-ROBUSTNESS-COARSE`（查「有没有 robustness 扫描、是不是二分定界」）。**两道门都过了**——因为 C2 **有**一段 `tolerance_source`、robustness **有**一次二分扫描。**但门查的是「在场」，不是「算对」**：来源字符串在场却算错了（漏 $A^2$ 因子），扫描在场却扫错了自由度（只扫 $e$、不扫 $A$）。**H2' 就活在「在场」与「算对」之间的缝里——这正是 P18 存在的全部理由。**
- **可否修**：① C2 容差按 $\bar h\propto A^2$ 求积重算（$\approx22\%$）；② `robustness_scan` 增扫 $A$（至少承认 $A$ 的 $10\%\to\bar h$ 的 $20\%$）与 $\tau_c$ 的 Model-1 预算；③ `eps_star` 用真实本底（$\sim22\%$）而非门宽报。
