## 模型

**安培模型**：均匀磁化圆柱 $\equiv$ 半径 $R$、长 $L$ 的**有限长螺线管**，侧面束缚面电流 $K = M_s$ (A/m)。

拆成一叠圆环，每环带电流 $dI = M_s\,dz'$，对 $z'\in[-L/2, L/2]$ 积分。单个圆环的场（Simpson 标准式，**只需第一、二类完全椭圆积分**）：

$$B_z = \frac{\mu_0 I}{2\pi\alpha^2\beta}\Big[(R^2-r^2-d^2)E(k^2) + \alpha^2 K(k^2)\Big],\qquad
B_r = \frac{\mu_0 I d}{2\pi\alpha^2\beta r}\Big[(R^2+r^2+d^2)E(k^2) - \alpha^2 K(k^2)\Big]$$

其中 $\alpha^2 = (R-r)^2+d^2$，$\beta^2 = (R+r)^2+d^2$，$k^2 = 4Rr/\beta^2$。

---

## ★ 与已有工作的连接：$B_r$ 就是我们已经有的那个闭式

由矢势 $\Phi = 2\pi r A_\phi$ 与 $B_r = -\partial A_\phi/\partial z$：

$$\boxed{\;B_r(r,z) = -\frac{1}{2\pi r}\frac{\partial \Phi}{\partial z}
= -\frac{M_s}{2\pi r}\Big[M(R,r,z+\tfrac{L}{2}) - M(R,r,z-\tfrac{L}{2})\Big]\;}$$

**这正是 `field.py` 里那个 Leibniz 化简的结果。** 于是：

- 管壁圆环的动生电动势 $\mathcal{E} = -v\,\partial_z\Phi = v\cdot 2\pi a\cdot B_r$
- **F-5 画的"涡流分布"，其实就是管壁上的径向磁场 $B_r$**

物理图像在这里闭合了：**驱动涡流的是径向场**。

## 三条独立的验证（**算之前写下的预期**）

| 门 | 预期 | 实测 | |
|---|---|---|---|
| **G-A** | 轴上必须回到教科书的有限长螺线管闭式解。**磁体中心 $B_z(0,0) = \dfrac{\mu_0 M_s (L/2)}{\sqrt{R^2+L^2/4}} = \dfrac{\mu_0 M_s}{\sqrt2} = 0.919$ T**（因 $R = L/2$） | 最大相对误差 **1.35e-14**；$B_z(0,0) = $ **0.9192 T** | ✅ |

| **G-C** | $B_r$ 的**两条完全不同的路**必须吻合：①把圆环的场积起来；②把互感对 $z$ 求导 | 最大相对误差 **5.13e-14** | ✅ |

**G-C 是最有分量的一条**：两条推导路径毫无共同之处，一条走 Biot–Savart + 椭圆积分，一条走互感 + Leibniz。它们吻合到 $5\times10^{-14}$——**这说明两边都对**。

## 结果

### S-1 · 真实的场，以及偶极子近似在哪里崩

**左**：$|\mathbf{B}|$ 与磁力线。**右**：$|\mathbf{B}_{\rm exact} - \mathbf{B}_{\rm dipole}| / |\mathbf{B}_{\rm exact}|$。

**管壁正好坐在误差 60–100% 的区域里。**

（磁体内部被挖掉了——点偶极子场在原点发散，那里的"误差"没有意义。）

### S-2 · 管壁上的 $B_r$ 与 $B_z$

| | 精确 | 点偶极子 |
|---|---|---|
| $B_r$ 的峰位 | **5.00 mm $= L/2$**（磁体端面） | 3.00 mm $= a/2$ |
| $B_z(a, 0)$ | $-186$ mT | $-376$ mT（**高估 2 倍**） |

**真实的场被磁体的有限长度"摊平"了，峰被推到端面上。** 这一条同时解释了三件事：为什么 $b$ 变小、为什么 $v_t$ 变大、为什么 F-5 的涡流峰在 $L/2$ 而不在 $a/2$。

### S-3 · 轴上的场

数值解与教科书闭式解**完全重合**。偶极子在 $z \lesssim 3L$ 处明显偏离，在管壁处**差 2 倍**。

### S-4 · 「$a \gg L$」到底是多少？

**A-1 的判据写作 $a \gg L$——但 "$\gg$" 不是一个数。这张图把它变成一个数。**

（全程用薄壁近似，因此**隔离出 A-1 一条**。）

| | |
|---|---|
| $v_t$ 要 **10%** 精度 | 需要 $a/L > 1.1$ |
| $v_t$ 要 **30%** 精度 | 需要 $a/L > 0.68$ |
| **本实验** | $a/L = 0.60$ —— **约差 2 倍** |

### ★ 两条假设的失效是**相乘**的，不是相加的

| | $b$ (N·s/m) | |
|---|---|---|
| 偶极子 + 薄壁（**Model-0**） | 2.10679 | |
| 有限磁体 + 薄壁 | 1.50558 | ← 放开 A-1 |
| 有限磁体 + 径向积分（**Model-2**） | 1.15293 | ← 再放开 A-2 |

$$\underbrace{1.3993}_{\text{A-1}} \times \underbrace{1.3059}_{\text{A-2}} = 1.8273 \;\Longrightarrow\; v_t \text{ 偏差 } +82.7\%$$

**与 `results.json` 里 AS-20 实测的 $v_t$ 偏差 $+82.7\%$ 逐位吻合。** 这是一次独立的自洽对拍——两条假设各自的效应，乘起来正好等于总效应。

> **注意别搞混**：S-4 的曲线在 $a/L = 0.60$ 处给出 **40%**，而 Model-2 vs Model-0 的 $v_t$ 偏差是 **82.7%**。**这两个数不该相等**——前者只含 A-1，后者含 A-1 × A-2。
